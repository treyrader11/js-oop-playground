1
Basic Class Creation
Create your first JavaScript class! This challenge introduces you to the fundamentals: constructors, properties, and methods.

Requirements:

  • Create a class called Book
  • Constructor should accept: title, author, pages
  • Method getInfo() returns: "Title by Author - Pages pages"
  • Method read() returns: "You are reading Title"
2
Class with Private Fields
Learn about encapsulation using private fields. Private fields start with # and can only be accessed from within the class.

Requirements:

  • Create a class called BankAccount
  • Use a private field #balance
  • Constructor accepts: accountHolder, initialBalance
  • Method deposit(amount) - adds to balance, returns new balance
  • Method withdraw(amount) - subtracts if sufficient funds, or returns "Insufficient funds"
  • Method getBalance() - returns current balance
  • Property accountHolder - stores account holder name
3
Basic Inheritance
Introduction to inheritance! Learn how to create child classes that extend parent classes and override methods.

Requirements:

  • Create parent class Animal with:
    • Constructor: name, species
    • Method makeSound(): "Some generic animal sound"
    • Method introduce(): "I am Name, a Species"
  • Create child class Dog that extends Animal:
    • Constructor: name, breed
    • Pass "Dog" as species to parent
    • Override makeSound(): "Woof! Woof!"
    • Add method getBreed() that returns the breed
4
Inheritance with Super
Master the super keyword! Learn how to call parent constructors and methods from child classes.

Requirements:

  • Create parent class Vehicle with:
    • Constructor: make, model, year
    • Method getAge(): returns vehicle age (2025 - year)
    • Method getInfo(): "Year Make Model"
  • Create child class ElectricCar that extends Vehicle:
    • Constructor: make, model, year, batteryCapacity
    • Use super() to call parent constructor
    • Override getInfo(): parent's info + " (Electric, XkWh battery)"
    • Add getBatteryInfo(): "Battery capacity: XkWh"
5
Multi-Level Inheritance & Polymorphism
Advanced inheritance! Create a three-level inheritance hierarchy and see polymorphism in action as different classes override the same methods.

Requirements:

  • Create parent class Shape with:
    • Constructor: name
    • Method getArea(): returns 0
    • Method getPerimeter(): returns 0
    • Method describe(): "This is a Name with area X and perimeter Y"
  • Create child class Rectangle that extends Shape:
    • Constructor: width, height
    • Pass "Rectangle" to parent
    • Override getArea(): width √ó height
    • Override getPerimeter(): 2 √ó (width + height)
  • Create grandchild class Square that extends Rectangle:
    • Constructor: side
    • Pass side for both width and height to parent
    • Override describe(): "This is a Square with side X, area Y and perimeter Z"
